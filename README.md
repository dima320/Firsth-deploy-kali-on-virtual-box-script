# Firsth-deploy-kali-on-virtual-box-script
Скрипт та інструкція до нього. Потрібно після першого запуску kali на віртуал бокс. Скрипт з'єднується з репозиторіями kali apt, оновлює систему, та встановлює основні додатки

1. завантажити файл kali-setup.sh на систему kali
2. зробити файл виконуваним chmod +x kali-setup.sh
3. запустити файл sudo ./kali-setup.sh

Зрозуміло що потрібно для виконання команд потрібно бути в одній папці з файлом kali-setup.sh інакше команди не спрацюють

або скрипт однією командою, можливо не працює потрібно буде протестити, перед використанням. Це можливо пригодиться в майбутньому
bash <(curl -s https://raw.githubusercontent.com/dima320/kali-setup/main/kali-setup.sh)
